My Foodie – Premium Food Delivery Web Application

My Foodie is a full-stack food ordering web application that allows users to browse food items, apply filters, manage cart and wishlist, and complete orders using multiple payment options.

The application provides a smooth and user-friendly food ordering experience with structured navigation and interactive UI.

Users can browse available food products and search items by name.
Products can be filtered by Vegetarian/Non-Vegetarian category and price range.
The cart system allows users to add, remove, and update item quantities.
Subtotal, delivery fee, and total amount are calculated dynamically.
Promo codes can be applied to receive discounts.

The authentication system includes login and logout functionality.
A modal-based login interface enhances user experience.
A personalized greeting appears inside the cart section.

The payment system supports Credit/Debit Card, UPI, and Cash on Delivery.
An order ID is generated after successful payment.
A confirmation screen is displayed upon successful transaction.

Users can view previously placed orders in the order history section.
The wishlist feature allows adding and removing favorite items.
Recommended products are displayed when the wishlist is empty.

The UI includes toast notifications and a loading spinner.
The layout is responsive and follows a clean modern design.

Technologies used include HTML5, CSS3, and JavaScript (Vanilla JS).
Backend integration can be done using Node.js, Express.js, and MongoDB.

The project demonstrates frontend design, client-side logic, and application flow management.
It is developed for learning, portfolio presentation, and full-stack practice
